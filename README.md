# Custom PHP & MySQL Portfolio Management System

A secure, dynamic Content Management System (CMS) built from scratch to showcase professional projects. This web application was developed and tested entirely using a mobile server environment (**KSWEB**), demonstrating practical problem-solving and resourceful full-stack capabilities.

## 🚀 Features
- **User Authentication:** Secure registration and login system with session management.
- **Dynamic CRUD Operations:** Admin dashboard to add, update, and delete projects seamlessly.
- **Advanced File Upload Security:** Implements rigorous image validation checking both file extensions and internal **MIME types** to block malicious scripts.
- **Modern Responsive UI:** Clean front-end layout featuring smooth CSS hover animations and transitions.

## 🛠️ Tech Stack
- **Backend:** PHP (OOP logic with PDO for secure database interactions)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3 (Modern grids & animations)
- **Environment:** KSWEB Mobile Server Suite (Apache/Nginx)
